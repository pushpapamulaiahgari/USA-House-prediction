# 🏠 USA House Price Predictor

End-to-End Machine Learning project to predict USA house prices using RandomForest Regression. Achieved R² score of 0.92 on test data.

## 🚀 Live Demo
https://usa-house-prediction-t25hfc6hthx7tzdw5prf5q.streamlit.app/

## 🛠️ Tech Stack & Results
- **Model**: RandomForest Regressor 
- **R² Score**: 0.92 | **MSE**: 1.00e+09
- **Libraries**: Scikit-learn, Pandas, Numpy, Joblib, Streamlit

## 📂 Project Structure
- `model_training.ipynb` : Complete pipeline with EDA, Feature Engineering, Training, and Model Evaluation
- `app.py` : Streamlit web application for real-time price prediction
- `model_trained.pkl` : Saved trained model file

## ▶️ How to Run Locally
```bash
git clone https://github.com/pushpapamulaiahgari/USA-House-prediction.git
cd HOUSE_prediction
pip install -r requirements.txt

# For Windows PowerShell
venv\Scripts\activate

streamlit run app.py
